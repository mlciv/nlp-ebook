# Syllabus

## Course Description

Natural language processing (NLP) is a core area of AI focused on modeling and generating human language. This course covers both foundations and modern deep learning approaches, including transformers and large language models.

## Learning Goals

- Understand core NLP tasks, data representations, and modeling choices
- Implement and train neural NLP models in PyTorch
- Evaluate model quality with rigorous metrics and error analysis
- Communicate experimental findings in technical writing and presentations

## Prerequisites

- Proficiency in Python programming
- Calculus and linear algebra
- Basic probability and statistics
- Introductory machine learning background
