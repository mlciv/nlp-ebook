# Final Project

The final project applies course methods to a substantial NLP task.

## Milestones

- **Proposal:** problem statement, related work, plan
- **Checkpoint:** progress report + preliminary results
- **Final Report:** experiments, analysis, conclusions
- **Presentation/Poster:** concise summary of the work

## Team Policy

- Team size: 1-3 students
- All team members should contribute meaningfully
- Cite external tools, models, and datasets used
- Clearly state each team member's contributions in final report
