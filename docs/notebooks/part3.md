# Part III — Transformers & LLMs

This part covers the architecture behind modern NLP — the **Transformer** — and how
**large language models** are adapted to specific tasks and turned into agents.

## Chapters in this part

- **[Chapter 4: Transformers & LLMs](hw4/Assignment_4.ipynb)** — Attention, the
  Transformer architecture, and working with pretrained models. Paired with an
  [illustrated, step-by-step walkthrough](hw4/Transformer_Illustration.ipynb).
- **[Chapter 5: Fine-Tuning & Agents](hw5/Part1-sft_trl_lora_qlora.ipynb)** — Supervised
  fine-tuning with parameter-efficient methods (LoRA/QLoRA via TRL), and
  [building an LLM agent](hw5/Part2-Exam_Agent_For_NLP.ipynb) with LangGraph.

## What you'll be able to do

- Explain self-attention and the Transformer block.
- Use pretrained Transformer models for downstream tasks.
- Fine-tune an LLM efficiently with LoRA/QLoRA.
- Compose an LLM into a simple agent.
